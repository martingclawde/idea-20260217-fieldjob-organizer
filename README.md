# idea-20260217-fieldjob-organizer

MVP ultra-simple (static) para organizar trabajos de campo tipo *dump trailer / junk removal / deliveries / service calls*.

- Sin login
- Sin backend
- Guarda en `localStorage`
- Deploy via GitHub Pages (template `martingclawde/perm-test`)

## Cómo correr

- Local: abrí `index.html` en el navegador.
- Online: GitHub Pages (ver URL en Settings → Pages).

## Qué hace

- Cargás trabajos (cliente, fecha/hora, duración, buffer, si requiere landfill y hora de cierre)
- Te da un chequeo rápido (salida estimada + warning si te pasás del cierre)
- Lista/agenda + export a texto para pegar en WhatsApp/Telegram

## Próximo paso de validación

- Mostrar a 5–10 personas que hacen servicios “en la calle” (junk removal, mudanzas, deliveries, etc.)
- Preguntar: ¿pagarías €5–10/mes por (a) recordatorios + (b) routing simple + (c) follow-ups automáticos?
